```javascript
Vue.component(Bill.name, Bill)
```

### API

### Bill Props
|属性 | 说明 | 类型 | 默认值 | 备注 |
|----|-----|------|------ |------|
|name|票据抬头|String| | |
|no|票据编号|String| | |
|water-mark|水印内容|String| |复杂内容使用`scoped slot`|

### Bill Slots

#### default
默认内容插槽

#### header
头部内容插槽

#### footer
底部内容插槽

#### watermask
水印内容scoped插槽

```html
<div slot="watermark" slot-scope="props">
  <!-- 水印内容 -->
</div>
```
